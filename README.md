# flappy-bird


Dette prosjektet bygger et spill som skal etterligne spillet Flappy Bird.

Prosjektets formål var det å leke seg med, og å bli komfortabel med bruk av pygame, samtidig som det ga god øving på objektorientert programmering. 

Prosjektet er skrevet i Python, med Pygame-biblioteket, og er påbegynt våren 2025.

For å teste koden selv kan det være nødvendig å installere pygame-biblioteket dersom det ikke allerede er på plass.
Etter det kan koden kjøres som hvilken som helst annet python-program, og spillvinduet vil dannes automatisk.
Spillet er i gang når det trykkes på mellomrom. Da vil fuglen hoppe, og spillet går ut på å få fuglen gjennom så mange hindringer som mulig uten å berøre hindringene. Da vil fuglen dø, og spillet er over.
